# 台球术语大全 (Billiards Terms Glossary)

这份文档汇集了台球（包括美式普尔 Pool、斯诺克 Snooker 等）的常见术语，旨在帮助初学者快速掌握相关概念。

## 1. 基础装备与球台 (Equipment & Table)

| 英文术语            | 中文术语           | 解释                                                           |
| :------------------ | :----------------- | :------------------------------------------------------------- |
| **Cue / Cue Stick** | 球杆               | 击球用的棍子，通常由前肢（Shaft）和后把（Butt）组成。          |
| **Cue Ball**        | 母球 / 白球        | 在比赛中唯一可以用球杆直接击打的白色球。                       |
| **Object Ball**     | 子球 / 目标球      | 被母球击打的球（如花式九球中的1-9号球，斯诺克中的红球/彩球）。 |
| **Pocket**          | 袋口               | 球台四周的六个洞口（底袋 Corner Pocket，中袋 Side Pocket）。   |
| **Rail / Cushion**  | 库边 / 岸          | 球台边缘的胶条部分，母球撞击后会反弹。                         |
| **Chalk**           | 巧粉 / 枪粉 / 擦粉 | 涂在皮头上的防滑粉，增加摩擦力，防止滑杆（Miscue）。           |
| **Tip**             | 皮头               | 球杆顶端接触母球的皮质部分。                                   |
| **Extension**       | 延长杆             | 加在球杆尾部增加长度的配件，用于击打远距离球。                 |
| **Rest / Bridge**   | 架杆               | 当手架无法够到母球时使用的辅助支架。                           |
| **Triangle / Rack** | 三角框 / 摆球纸    | 用于开局时将子球摆放整齐的工具。                               |
| **Nap**             | 顺毛 / 逆毛        | 台尼的纹理方向（斯诺克台尼有明显的顺逆毛，影响球速和走位）。   |

## 2. 击球与杆法 (Techniques & Spin)

| 英文术语                  | 中文术语           | 解释                                                                                             |
| :------------------------ | :----------------- | :----------------------------------------------------------------------------------------------- |
| **Stance**                | 站位               | 击球时的身体姿势。                                                                               |
| **Bridge**                | 手架               | 击球时放在球台上支撑球杆的前手姿势（分开放式 Open Bridge 和封闭式 Closed/Loop Bridge）。         |
| **Stroke**                | 出杆               | 运杆和击打的整个动作过程。                                                                       |
| **Follow Shot / Topspin** | 高杆 / 推杆        | 击打母球中上部，使母球在撞击子球后继续向前滚动。                                                 |
| **Draw Shot / Backspin**  | 低杆 / 缩杆 / 拉杆 | 击打母球中下部，使母球在撞击子球后向后旋转（回退）。                                             |
| **Stun Shot**             | 定杆               | 击打母球中心位置（或略低），使母球撞击子球后几乎停留在原位（如果包含分离角，则沿切线方向移动）。 |
| **English / Side Spin**   | 加塞 / 侧旋        | 击打母球的左侧或右侧，使母球产生旋转，主要影响吃库后的反弹角度。                                 |
| **Masse**                 | 扎杆               | 竖起球杆击打母球，使母球产生强烈弧线运动。                                                       |
| **Jump Shot**             | 跳球               | 击打母球下部或使用特制跳杆，使母球跳起越过障碍球。                                               |
| **Stop Shot**             | 定球               | 纯粹的定杆，母球撞击子球后完全静止（通常用来作为练习）。                                         |

## 3. 游戏规则与过程 (Gameplay)

| 英文术语                | 中文术语            | 解释                                                                                                                   |
| :---------------------- | :------------------ | :--------------------------------------------------------------------------------------------------------------------- |
| **Break**               | 开球                | 比赛开始的第一杆击球。                                                                                                 |
| **Dry Break**           | 违例开球 / 无球落袋 | 开球后没有子球入袋（美式规则中常导致交换球权）。                                                                       |
| **Scratch**             | 洗袋 / 母球落袋     | 母球掉进袋口，通常属于犯规。                                                                                           |
| **Foul**                | 犯规                | 违反规则的行为（如连击、未碰到目标球、母球洗袋等）。                                                                   |
| **Ball in Hand**        | 自由球              | 对手犯规后，母球可以放在台面上任意位置进行击球（主要见于美式普尔）。                                                   |
| **Free Ball**           | 自由球              | 斯诺克特有规则，对手犯规造成斯诺克后，可指定任意彩球当作红球击打。                                                     |
| **Safety**              | 防守                | 战术性击球，不以进球为主要目的，而是将母球停在对手难以进攻的位置。                                                     |
| **Snookered**           | 被做斯诺克 / 障碍球 | 母球的直线行进路线上有障碍球阻挡，无法直接击打到目标球。                                                               |
| **Clearance / Run Out** | 清台 / 一杆清       | 连续击打落袋直到赢下该局。                                                                                             |
| **Break and Run**       | 炸清                | 开球进球后直接一杆清台，对手没有上场机会。                                                                             |
| **Push Out**            | 推杆 / 如果         | 九球规则特有，开球后如果局面不佳，一方可选择打一杆安全球（不需碰库或碰球），如果你选择推杆，对手可以选择打或归还球权。 |

## 4. 战术与常见俚语 (Strategy & Slang)

| 英文术语                | 中文术语        | 解释                                                                           |
| :---------------------- | :-------------- | :----------------------------------------------------------------------------- |
| **Bank Shot**           | 翻袋            | 母球撞击子球，使子球先撞库边再反弹入袋。                                       |
| **Kick Shot**           | 解球 / 反弹球   | 母球先撞库边，再去击打目标球（通常用于解斯诺克）。                             |
| **Combination / Plant** | 组合球 / 传球   | 用母球撞击A球，A球再去撞击B球入袋。                                            |
| **Carom / Cannon**      | 借球 / 撞击     | 母球撞击目标球后，利用分离角去撞击另一颗球（开伦台球的核心玩法，或用于解球）。 |
| **Double Hit**          | 连击            | 杆头接触母球两次，属犯规。                                                     |
| **Miscue**              | 滑杆            | 球杆打呲了，通常是因为巧粉不够或击球点过偏。                                   |
| **Squirt / Deflection** | 让点 / 偏移     | 加塞击球时，母球瞬间偏离瞄准线的现象（修正让点是进阶必修课）。                 |
| **Throw**               | 抛 / 甩         | 母球与子球碰撞时的摩擦力导致子球偏离切线方向的现象（特别是加塞或齿轮效应）。   |
| **Frozen Ball**         | 贴库球 / 贴死球 | 球紧贴着库边或另一颗球。                                                       |
| **Fluke**               | 运气球 / 蒙进   | 并非故意瞄准但意外进球。                                                       |
| **Pocket Speed**        | 袋口速度        | 控制力度，使子球刚好掉入袋口，或不进时停在袋口阻挡对手。                       |
| **Thin Cut**            | 薄球            | 极薄地切击子球边缘。                                                           |
| **Pot**                 | 进球            | 斯诺克常用词，意为将球打入袋中。                                               |

## 5. 常见比赛项目

- **8-Ball (中式八球 / 美式八球):** 双方分全色（1-7）和花色（9-15），最后打进8号球获胜。
- **9-Ball (九球):** 按照1-9顺序击打，打进9号球者获胜（可利用组合球提前打进）。
- **Snooker (斯诺克):** 击打红球与彩球交替，分值最高，对精准度和防守要求极高。
- **Chinese 8-Ball (中式台球):** 使用类似斯诺克的球台（袋口小、库性不同）打美式八球的规则。
