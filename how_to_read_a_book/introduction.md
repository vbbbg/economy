# 如何阅读一本书 (How to Read a Book)

本书由莫提默·J·艾德勒 (Mortimer J. Adler) 和查尔斯·范多伦 (Charles Van Doren) 所著，是一本关于阅读技巧的经典指南。

## 学习目标

1. 理解阅读的四个层次：基础阅读、检视阅读、分析阅读、主题阅读。
2. 掌握不同类型书籍的阅读策略（尤其是论说性书籍）。
3. 提高主动阅读（Active Reading）的能力，变被动接受为主动思考。

## 阅读的四个层次

阅读并非整齐划一的行为，根据目的和深度，可以分为以下四个层次：

1. **基础阅读 (Elementary Reading)**：解决“这句话在说什么？”的问题。
2. **检视阅读 (Inspectional Reading)**：解决“这本书在谈什么？”的问题。
3. **分析阅读 (Analytical Reading)**：解决“这本书在详细谈什么，以及是否有道理？”的问题。
4. **主题阅读 (Syntopical Reading)**：解决“关于这个主题，这些书共同说了什么？”的问题。
